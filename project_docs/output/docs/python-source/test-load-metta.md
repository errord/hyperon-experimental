# test_load.metta

## 文件角色

**解析器/加载** 行为的fixture：多行 S 表达式，覆盖空格、嵌套、负数与浮点字面量等 token 形态。

## 原子分类

- **符号**：`A`…`F`。
- **字面量**：`-a`、`-1.1`、`1`。
- **嵌套表达式**：多行缩进括号结构。

## 关键运算/函数

无 `!` 命令；纯数据面，由 `load`/`parse` 测试读取。

## 演示的 MeTTa 特性

- 行首 `A B` 与 `()(A B)` 等检验**空元与序列**分界。
- 多行列表检验 **换行与缩进** 的鲁棒性。

## 教学价值

作为**文法与读取路径**的黑盒输入，比教程脚本更偏工程测试。

## 概要

六行左右由浅入深的文本结构，用于验证加载与原子序列化一致性。
