# f1_moduleC.metta

## 文件角色

模块依赖图的**公共叶节点**：导出带辅助常量的 `g`，供 moduleA 与 moduleB 共享，形成钻石依赖。

## 原子分类

- **内部常量**：`__g`（零元，`(: __g (-> Number))`）。
- **导出函数**：`g`：`Number -> Number`。

## 关键运算/函数

`(= (__g) 100)`；`(= (g $x) (+ $x (__g)))`。

## 演示的 MeTTa 特性

- 小模块只暴露**数值偏移**语义，使 `f`、`dup` 测试断言可预期（如 `g 2` → 102）。
- 两路径导入 C 时应**不重复**破坏该语义（由 f1_imports 验证）。

## 教学价值

演示如何把**共享库**与**政策各异的上层模块**分离。

## 概要

定义 `__g`=100 与 `g x = x + 100`，作为 A/B 的共同依赖。
