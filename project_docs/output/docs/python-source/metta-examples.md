# metta_examples.py（仓库根）

## 文件角色

**中文注释的 MeTTa 能力长演示脚本**：用 `MeTTa` + `Environment.custom_env` 分章节打印 DAG、超图、CRUD、公司知识图谱上的联合 `match`、以及函数/类型/superpose/推理/State/let/case/列表原语等。

## 关键特性与集成

- **stdout**：`utf-8` 包装便于中文输出。
- **无扩展注册**：纯标准库式 MeTTa 程序字符串 + `run`。
- **体量**：单文件教学/演讲材料级别。

## 摘要

面向读者的可运行文档，展示 MeTTa 作为图与逻辑语言的用法；非库代码，适合入门与回归人工走查。
