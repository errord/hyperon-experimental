# sandbox：resolve

**角色**：嵌套 `MeTTa`（`r`）与 `::` 限定调用的语义探针。  
**要点**：`r::fact`、`r::self-from-self`、`match`、`r::call_func`。  
**概要**：对比 `&self` 与 `r::&self` 上 `content` 的匹配差异；`r::content` 为 OperationAtom；讨论 `::` 非严格命名空间。
