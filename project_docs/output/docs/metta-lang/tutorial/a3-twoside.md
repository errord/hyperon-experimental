# a3_twoside.metta

## 文件角色

进阶符号匹配教程：规则两侧变量、合取查询 `,`、以及“部分变量不绑定”的查询语义。

## 原子分类

- **规则与事实**：`implies`、`Frog`、`Robot` 等。
- **变量**：`$x`、`$y`、`$P`、`$what`。
- **合取**：`,` 连接多个子查询并共享变量绑定。

## 关键运算/函数

`match`、`assertEqualToResult`；合取模式 `(, ...)`。

## 演示的 MeTTa 特性

- 模式两侧变量同时替换（蕴含前件/后件）。
- 用 `,` 做**共享变量的合取**，模拟简单前向/联合查询。
- 允许查询中部分变量保持未绑定，得到概括性答案。

## 教学价值

从“单模式匹配”过渡到**小型逻辑程序式**的 `match` 用法，为反向链教程铺垫。

## 概要

用 `implies` 与事实 `Frog Sam` 等展示合取查询如何推出多重结论，以及如何用未绑定变量表达“那么一定是 Frog”这类元答案。
